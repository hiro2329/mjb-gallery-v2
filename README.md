# 📸 MJB Photo Gallery (Ver.2)

여행의 순간들을 기록하는 개인 사진 갤러리 프로젝트입니다.
기존 정적 웹사이트 방식의 한계를 느끼고, 유지보수와 확장성을 고려하여 **새롭게 리뉴얼**하고 있습니다.

## 🎯 프로젝트 개요

이 프로젝트는 단순한 디자인 변경이 아니라, **시스템적인 문제를 해결**하기 위해 시작되었습니다.

### ✅ 리뉴얼 목표 (To-Be)

- **Supabase(Backend)** 를 도입하여 데이터와 이미지를 효율적으로 관리.
- **Admin(관리자) 페이지**를 구축하여 코드를 건드리지 않고도 사진 업로드 가능.
- **React**와 **TypeScript**를 사용하여 더 안정적이고 유지보수하기 좋은 구조로 개편.

### 🚧 Current Status
